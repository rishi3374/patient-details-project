A professional healthcare dashboard for patient management, diagnosis tracking, and medical records. Built from scratch with React, TypeScript, Vite, Tailwind CSS, and shadcn-ui.

## Features

- Patient overview and details
- Appointments and diagnostics
- Charts and analytics
- Responsive UI with dark mode

## Tech Stack

- Vite (React + TypeScript)
- Tailwind CSS + shadcn-ui
- Radix UI primitives
- React Router

## Getting Started

Prerequisites:
- Node.js (LTS) and npm. Recommended install via nvm: https://github.com/nvm-sh/nvm

Install and run:

```sh
npm ci            # install dependencies from lockfile
npm run dev       # start local dev server (http://localhost:8080)
```

Build and preview:

```sh
npm run build     # production build
npm run preview   # preview the production build
```

## Project Scripts

- dev: Start Vite dev server
- build: Build for production
- build:dev: Development-mode build
- preview: Preview production build locally
- lint: Lint the codebase
